[click here to invite!](https://github.com/ronnapatp/antibadwordbot)
# Install
1. Install dependencies
``` shell
$ npm install
```
2. Change `.env.example` to `.env` and insert token
``` diff
# .env file
- DISCORD_TOKEN=
+ DISCORD_TOKEN=Your token
```
3. Run bot
``` shell
$ node .
```
